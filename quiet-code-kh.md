
# សិល្បៈនៃកូដដ៏ស្ងប់ស្ងាត់

នៅក្នុងការដេញតាមមិនឈប់ឈរនៃ "ភាពលើសលប់"—មុខងារកាន់តែច្រើន ល្បឿនកាន់តែលឿន—យើងបានភ្លេចនូវតម្លៃពីកំណើតនៃភាពស្ងប់ស្ងាត់។ ក្នុងវិស្វកម្ម ភាពស្ងប់ស្ងាត់បង្ហាញតាមរយៈភាពសាមញ្ញ។

### ទម្ងន់នៃ Abstraction
រាល់ស្រទាប់ abstraction ដែលយើងបន្ថែម គឺជាការសន្យាដែលយើងធ្វើចំពោះអនាគត។ ការសន្យាទាំងនេះភាគច្រើនត្រូវបានរំោភ។ យើងបង្កើតប្រព័ន្ធស្មុគស្មាញដើម្បីដោះស្រាយបញ្ហាដែលប្រហែលជាមិនដែលកើតមាន។

> "ភាពល្អឥតខ្ចោះត្រូវបានសម្រេចបាន មិនមែននៅពេលដែលគ្មានអ្វីត្រូវបន្ថែមទៀតនោះទេ ប៉ុន្តែនៅពេលដែលគ្មានអ្វីនៅសល់ដើម្បីដកចេញ។" — Antoine de Saint-Exupéry

### ការអនុវត្តវិស្វកម្មបែប Minimalist
1. **សម្លាប់អំនួត។** កុំសរសេរកូដដែលឆ្លាតពេក ត្រូវសរសេរកូដដែលច្បាស់លាស់។
2. **ទទួលយកការរឹតត្បិត។** ការរឹតត្បិតមិនមែនជាជញ្ជាំងទេ វាគឺជាផ្ទាំងក្រណាត់។
3. **តម្លៃនៃភាពធន់។** តើកូដនេះនឹងនៅតែមានន័យទេ នៅពេលដែលវដ្តនៃភាពល្បីល្បាញបានកន្លងផុតទៅ?
